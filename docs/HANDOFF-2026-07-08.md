## Engineering gotchas (verified this session)

- **Migrations: plain SQL only** — no DO $$ / nested dollar quoting (drizzle-kit
  dies silently). packages/db/scripts/migrate.mjs is the migrate path. Latest
  migration: 0022.
- **Kimi is a thinking model**: reasoning_content eats tokens; content empty at
  tiny max_tokens. The brain's output contract now handles this, but remember it
  for any new LLM surface. LLM aliases: cheap/analysis/reasoning/code → Moonshot,
  embed → OpenAI. Config: infra/litellm.config.yaml (repo = source of truth for
  the staging stack, synced by deploy).
- **litellm publishes no host port** — reachable only inside the compose network
  (`http://litellm:4000`). Probe from inside companyos-os-prod (node fetch, env
  has the virtual keys).
- **Cloudflare 524s at 100s**: long API calls (brain runs) must hit
  `http://127.0.0.1:3000` on the VPS, not the public URL. Runs continue
  server-side after a 524 anyway.
- **MCP endpoint accepts direct JSON-RPC** (stateless): single POST
  `{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{...}}` with Bearer
  cos_ token to https://cos-staging.risi.au/api/mcp.
- **Wizard content byte-parity**: docs/tasks/M8-07-content/* == DEFAULT_TEMPLATE_
  FILES in intake/service.ts == skills repo. Synced skills_index rows SHADOW code
  defaults — if wizard behavior looks stale after a content change, check which
  copy is winning and sync.
- **PowerShell 5.1**: no `&&`; embedded double quotes in `git commit -m`
  here-strings break arg parsing (message becomes pathspecs — commit fails but
  subsequent commands still run; check output). Prefer the Bash tool for git+gh.
- **Windows/MSYS**: MSYS_NO_PATHCONV=1 for docker exec path args; clean
  apps/os/.next after branch switches.
- **VPS**: stack at ~/app, podman with docker-compose provider; postgres is
  pgvector/pgvector:pg17; DBs: companyos, litellm (NO plane DB on staging);
  POSTGRES_USER in .env. `.env` keys of note: MOONSHOT_API_KEY, OPENAI_API_KEY,
  LITELLM_EMBED_KEY, BRAIN_LITELLM_API_KEY, BRAIN_ENGINE_TOKEN, GITHUB_TOKEN
  (PAT), GITHUB_ORG, SKILLS_REPO, GITHUB_WEBHOOK_SECRET, COMPOSE_PROFILES=brain,
  (pending) COS_VAULT_KEY. Backup exists: .env.bak-2026-07-07.
